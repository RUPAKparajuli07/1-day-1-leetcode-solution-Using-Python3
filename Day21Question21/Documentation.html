Merge Two Sorted Lists

Status: Solved

Difficulty: Easy

Topics

  • Linked Lists
  • Merging

Companies

  • Various

Description

You are given the heads of two sorted linked lists list1 and list2. Merge the two lists into one sorted list. The list should be made by splicing together the nodes of the first two lists. Return the head of the merged linked list.

Example 1

Input: list1 = [1,2,4], list2 = [1,3,4]

Output: [1,1,2,3,4,4]

Example 2

Input: list1 = [], list2 = []

Output: []

Example 3

Input: list1 = [], list2 = [0]

Output: [0]

Constraints

  • The number of nodes in both lists is in the range [0, 50].
  • -100 <= Node.val <=100
  • Both list1 and list2 are sorted in non-decreasing order.

Algorithm

1. Create a dummy node to start the merged list.

2. Initialize a pointer to keep track of the current node in the merged list.

3. Iterate through both lists until one of them becomes None.

4. Compare the values of the current nodes of list1 and list2.

5. Connect the smaller value node to the merged list.

6. Move the pointer of the merged list forward.

7. Connect the remaining nodes of the non-empty list to the merged list.

8. Return the next node of the dummy, which is the head of the merged list.
