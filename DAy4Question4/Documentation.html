Median of Two Sorted Arrays - Documentation

Problem Statement:

Given two sorted arrays nums1 and nums2 of size m and n respectively, find the median of the two sorted arrays.

Approach:

The problem can be solved using a binary search approach. We find the kth smallest element in the combined array of nums1 and nums2. If the total length of both arrays is odd, we return the kth element directly. If it's even, we return the average of the kth and (k+1)th elements.

Complexity:

The time complexity of the solution is O(log(min(m, n))) where m and n are the lengths of nums1 and nums2 respectively.

Algorithm:

1. Calculate the total length of the combined arrays, `total_len = len(nums1) + len(nums2)`.

2. Determine whether the total length is odd or even:

  • If it's odd, the median will be the single middle element.
  • If it's even, the median will be the average of the two middle elements.

3. Define a helper function `find_kth(nums1, nums2, k)` to find the kth smallest element in the combined arrays:

  • Ensure `nums1` is smaller or equal in length to `nums2`.
  • Handle base cases.
  • Divide `k` into two parts and search recursively.

4. Call the `find_kth` function to find the median element(s).
