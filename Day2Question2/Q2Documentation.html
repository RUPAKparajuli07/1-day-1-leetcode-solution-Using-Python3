Add Two Numbers in a Linked List

This code solves the "Add Two Numbers" problem where two non-empty linked lists are provided, each representing a non-negative integer. The digits of each integer are stored in reverse order, and the task is to add the two numbers represented by these linked lists and return the sum as a linked list.

Approach

The solution traverses both input linked lists simultaneously while keeping track of the carry from the previous addition. It iterates through the linked lists until both are exhausted and there is no carry left. In each iteration, it calculates the sum of corresponding digits along with the carry, creates a new node with the sum digit, and advances to the next nodes in the input lists. Finally, it returns the resulting linked list representing the sum.

Pseudocode

  1. Initialize a dummy head node for the resulting linked list.
  2. Initialize a current pointer to the dummy head.
  3. Initialize a carry variable to store the carry from the previous addition.
  4. Iterate through the input linked lists while there are nodes left in either list or there is a carry:
    1. Extract the values of the current nodes in both input lists (or 0 if the list is exhausted).
    2. Calculate the sum of the values along with the carry.
    3. Update the carry for the next iteration.
    4. Create a new node with the digit sum % 10 and append it to the resulting linked list.
    5. Move to the next nodes in the input lists if they exist.
  5. Return the next node after the dummy head, which contains the resulting linked list representing the sum.

Complexity Analysis

  • Time Complexity: O(max(n, m)), where n and m are the lengths of the input linked lists l1 and l2, respectively. The algorithm traverses through both linked lists once.
  • Space Complexity: O(max(n, m)), the space required for the resulting linked list, where n and m are the lengths of l1 and l2, respectively.
