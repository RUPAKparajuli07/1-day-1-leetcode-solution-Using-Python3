Merge k Sorted Linked Lists

This Python code implements a solution for merging k sorted linked lists using the merge sort algorithm.

Algorithm Explanation:

  1. merge(self, left: ListNode, right: ListNode) -> ListNode: This function merges two sorted linked lists left and right into one sorted linked list. It iterates through both lists, comparing elements, and appending the smaller element to the merged list until one of the lists is exhausted. Then, it appends the remaining elements from the non-empty list to the merged list.
  2. mergeSort(self, lists: List[ListNode], start: int, end: int) -> ListNode: This function implements the merge sort algorithm recursively. It divides the list of linked lists into halves until the base case, where there's only one list left. Then, it merges the lists back together by calling the merge function.
  3. mergeKLists(self, lists: List[ListNode]) -> ListNode: This is the main function that takes a list of sorted linked lists as input and returns a single sorted linked list. It first checks if the input list is empty and returns None if so. Otherwise, it calls the mergeSort function to merge the lists together.

This implementation efficiently handles the merging of k sorted linked lists by utilizing the merge sort algorithm, which has a time complexity of O(n log k), where n is the total number of elements across all lists and k is the number of lists.
