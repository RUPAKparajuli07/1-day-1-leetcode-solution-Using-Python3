Palindrome Number - Documentation

Question:

Given an integer x, return true if x is a palindrome, and false otherwise.

Algorithm:

To determine if an integer is a palindrome:

  1. If the integer is negative, return false.
  2. Reverse the integer.
  3. If the reversed integer is equal to the original integer, return true; otherwise, return false.
