3Sum Closest Documentation

Description

This function finds three integers in a given array such that their sum is closest to a target integer.

Function Signature

def threeSumClosest(self, nums: List[int], target: int) -> int:

Parameters

  • nums: List of integers - The input array.
  • target: Integer - The target sum.

Returns

Returns the sum of the three integers that is closest to the target.

Example

Input: nums = [-1, 2, 1, -4], target = 1
Output: 2
Explanation: The sum that is closest to the target is 2. (-1 + 2 + 1 = 2).

Algorithm

1. Sort the input array nums.
2. Initialize closest_sum to positive infinity.
3. Iterate through the array from index 0 to len(nums) - 3:
    4. Initialize left pointer to i + 1 and right pointer to len(nums) - 1.
    5. While left pointer is less than right pointer:
        6. Calculate the current_sum as nums[i] + nums[left] + nums[right].
        7. If current_sum is equal to target, return target.
        8. Update closest_sum if the absolute difference between current_sum and target is smaller than the absolute difference between closest_sum and target.
        9. Move pointers:
            - If current_sum is less than target, move left pointer to the right.
            - If current_sum is greater than target, move right pointer to the left.
10. Return closest_sum.
