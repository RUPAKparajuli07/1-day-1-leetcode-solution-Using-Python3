Method: lengthOfLongestSubstring

This method takes a string 's' as input and returns an integer representing the length of the longest substring without repeating characters.

Parameters:

  • s: A string for which the length of the longest substring without repeating characters needs to be found.

Returns:

An integer representing the length of the longest substring without repeating characters.

Algorithm:

  1. Initialize a dictionary 'char_index' to store the index of the last occurrence of each character.
  2. Initialize 'max_length' to 0 and 'start' to 0.
  3. Iterate through the string using a sliding window defined by 'start' and 'end' pointers.
  4. If the current character is already in 'char_index' and its index is after the start of the current substring, update the 'start' pointer to the index after the last occurrence of this character.
  5. Update the index of the current character in 'char_index'.
  6. Update 'max_length' if needed by comparing the length of the current substring with the maximum length found so far.
  7. Return 'max_length' as the result.
