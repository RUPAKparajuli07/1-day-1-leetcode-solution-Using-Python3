Reverse Integer Documentation

This documentation explains the implementation of a Python class Solution for reversing an integer.

Algorithm:

  1. Initialize a variable sign to 1 if x is greater than or equal to 0, otherwise -1.
  2. Take the absolute value of x to handle the sign.
  3. Convert x to a string, reverse it, and convert it back to an integer.
  4. Check if the reversed integer is within the 32-bit range (-2^31 to 2^31 - 1).
  5. If it is within the range, return the reversed integer; otherwise, return 0.

Example Usage:

Example 1:
Input: 123
Output: 321

Example 2:
Input: -123
Output: -321

Example 3:
Input: 120
Output: 21
