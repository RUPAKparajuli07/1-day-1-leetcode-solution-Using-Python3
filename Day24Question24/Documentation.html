Swap Nodes in Pairs Documentation

Description

Given a linked list, swap every two adjacent nodes and return its head. The values in the list's nodes are not modified, only the nodes themselves may be changed.

Examples

Example 1:

Input: head = [1,2,3,4]
Output: [2,1,4,3]

Example 2:

Input: head = []
Output: []

Example 3:

Input: head = [1]
Output: [1]

Constraints

  • The number of nodes in the list is in the range [0, 100].
  • 0 <= Node.val <=100

Algorithm

To solve this problem, we traverse the linked list while swapping pairs of adjacent nodes. We keep track of three nodes at a time: the current node, the next node, and the node after the next node. Then, we perform the swapping operation by adjusting the pointers accordingly.

If the linked list has less than two nodes, no swapping is needed. We use a dummy node to handle edge cases. Then, we iterate through the list, swapping pairs of adjacent nodes until we reach the end.
