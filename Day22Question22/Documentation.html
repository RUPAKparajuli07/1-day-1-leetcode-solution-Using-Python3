Generate Parentheses Documentation

Class: Solution

Description: This class provides a method to generate all combinations of well-formed parentheses for a given number of pairs.

Method: generateParenthesis

Parameters:

  • n: int - The number of pairs of parentheses.

Return Value: List[str] - A list containing all combinations of well-formed parentheses.

Example 1:

Input: n = 3

Output: ["((()))","(()())","(())()","()(())","()()()"]

Example 2:

Input: n = 1

Output: ["()"]

Constraints:

  • 1 <= n <=8

Algorithm:

  1. Define a recursive function backtrack.
  2. If the length of the current string s is equal to 2 * n, add s to the result list.
  3. If the count of left parentheses added left is less than n, recursively call backtrack with s + '(' and increment left.
  4. If the count of right parentheses added right is less than left, recursively call backtrack with s + ')' and increment right.
  5. Initialize an empty list result and call backtrack() without any arguments.
  6. Return the result list.
