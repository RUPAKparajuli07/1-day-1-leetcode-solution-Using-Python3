3Sum Function Documentation

Description

The threeSum function is designed to solve the classic 3Sum problem. Given an integer array, it returns all unique triplets from the array where the sum of the triplet equals zero.

Parameters

  • nums: An array of integers. It represents the input array for which the 3Sum problem needs to be solved.

Return Value

The function returns a list of lists, where each inner list contains three integers that sum up to zero. These lists represent unique triplets found in the input array.

Examples

Example 1:

            Input: nums = [-1,0,1,2,-1,-4]
            Output: [[-1,-1,2],[-1,0,1]]

Example 2:

            Input: nums = [0,1,1]
            Output: []

Example 3:

            Input: nums = [0,0,0]
            Output: [[0,0,0]]

Algorithm

The algorithm used by the threeSum function employs a two-pointer approach along with sorting the input array. It iterates through the array, skipping duplicates, and for each element, it uses two pointers to find the other two elements whose sum equals zero.
