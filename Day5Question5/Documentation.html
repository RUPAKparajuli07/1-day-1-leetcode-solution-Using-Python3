Longest Palindromic Substring Algorithm

Algorithm:

The algorithm for finding the longest palindromic substring in a given string involves using dynamic programming.

  1. Initialize a 2D table to store whether substrings are palindromic.
  2. Iterate over all substrings of length 1 and mark them as palindromic.
  3. Iterate over all substrings of length 2 and mark palindromic if the characters at both ends are the same.
  4. Iterate over substrings of length 3 or more, checking if the characters at both ends are the same and if the substring between them is palindromic.
  5. Keep track of the longest palindromic substring found.
  6. Return the longest palindromic substring.

Example:

Let's see an example of how to use the algorithm:

Input: s = "babad"

Output: "bab" or "aba"

Input: s = "cbbd"

Output: "bb"
