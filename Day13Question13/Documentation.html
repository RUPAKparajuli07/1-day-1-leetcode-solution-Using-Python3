Roman to Integer Conversion

Problem Description

Roman numerals are represented by seven different symbols: I, V, X, L, C, D, and M. The goal is to convert a Roman numeral string to an integer.

Algorithm

    class Solution:
        def romanToInt(self, s: str) -> int:
            roman_map = {
                'I': 1,
                'V': 5,
                'X': 10,
                'L': 50,
                'C': 100,
                'D': 500,
                'M': 1000
            }
            
            total = 0
            prev_value = 0
            
            for char in s:
                current_value = roman_map[char]
                if current_value > prev_value:
                    total += current_value - 2 * prev_value
                else:
                    total += current_value
                prev_value = current_value
            
            return total

Documentation

This algorithm takes a Roman numeral string as input and iterates through each character. It maintains a running total and a previous value to handle cases where subtraction is needed. If the current value is greater than the previous value, it subtracts twice the previous value from the current value to account for the previous addition. Otherwise, it simply adds the current value to the total. Finally, it returns the total integer value.
