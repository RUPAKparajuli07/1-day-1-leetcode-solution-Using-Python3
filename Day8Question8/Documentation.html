String to Integer (atoi) - Documentation

The task is to implement the myAtoi(string s) function, which converts a string to a 32-bit signed integer, similar to C/C++'s atoi function.

Problem Statement:

Implement the myAtoi(string s) function, which converts a string to a 32-bit signed integer according to the following algorithm:

Algorithm:

  1. Ignoring Leading Whitespace:
    • Initialize a pointer i to 0.
    • Iterate through the string s until i reaches the end or encounters a non-whitespace character.
    • Increment i until a non-whitespace character is found.
  2. Checking for Sign:
    • Check if the character at position i is '-' or '+'.
    • If '-', set sign to -1; if '+', set sign to 1.
    • Increment i accordingly.
  3. Converting Digits:
    • Initialize num to 0.
    • Iterate through the string starting from position i.
    • While the character at i is a digit:
      • Convert the character to an integer and add it to num after multiplying num by 10.
      • Increment i.
  4. Clamping to 32-bit Signed Integer Range:
    • Multiply num by sign.
    • If num is less than -2^31, return -2^31.
    • If num is greater than 2^31 - 1, return 2^31 - 1.
    • Otherwise, return num.
